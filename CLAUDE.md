# Claude Code Context

## Repository Map

- `gerrywolfe.com` is served from **`crywolfe/crywolfe.github.io`** (this repo)
- `lambda.gerrywolfe.com` is served from **`crywolfe/lambda-engine`**

## Deployment

- GitHub Pages serves from the **`master`** branch (not `main`)
- After pushing to `master`, Pages takes 1–2 minutes to deploy
- If changes don't appear, do a hard refresh (`Cmd+Shift+R` / `Ctrl+Shift+R`) to bypass browser cache
